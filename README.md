
About the Game

Genre: Arcade / Casual
Reference: Classic Snake + Pac-Man

The game is a hybrid of the classic Snake and Pac-Man concepts.
The player controls a growing snake that navigates through a maze, eats dots, and avoids enemies.

Core idea: eat and grow while surviving in a hostile maze.

Controls
Swipe gestures (up, down, left, right)
The snake moves continuously in the chosen direction
Player changes direction using swipes
⚙️ Core Mechanics
The player controls a snake moving through a maze
The goal is to eat all dots in the level
Each eaten dot may increase the snake’s length
There are ghost enemies that:
Move through the maze
Chase the player
The player must avoid collision between the snake’s head and ghosts
Collision with a ghost = lose condition
MVP Scope (Minimum Viable Product)

The first playable prototype should include:

Gameplay
One playable maze level
Basic snake movement (grid-based)
Swipe input controls
Collectible dots placed in the maze
Snake growth mechanic (optional for MVP, but preferred)
Enemies
At least one ghost enemy
Basic AI behavior:
Random movement OR simple chasing logic
Game States
Start screen (minimal)
Gameplay screen
Lose condition (collision with ghost)
Win condition (all dots collected)
UI
Score counter (optional but recommended)
Simple restart button
Definition of Done (Acceptance Criteria)

The MVP is considered complete when:

The player can:
Control the snake using swipe gestures
Move smoothly within the maze
All dots can be collected
Ghosts are present and interact with the player
Collision with a ghost correctly triggers a game over
Collecting all dots triggers a win condition
The game can be restarted without bugs
No critical crashes or blocking issues occur during gameplay
